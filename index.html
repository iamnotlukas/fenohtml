Tags

As tags são propriedades que compoem uma linguagem de programação, sendo ela HTML, Javascript, C#, CSS, etc. Cada tag que é usada em uma determinada linguagem tem uma função, e seus atributos, Podemos dar exemplo a tag "h6" que tem como um um dos principais objetivos criar um texto para a leitura do usuário. Dependendo da tag ela pode conter diversos atributos que se pode utilizar, e cada atributo tem uma função na linguagem usada

Linguagem

Linguagem da programação é uma linguagem usada para a interpretação de um programa ara vizualização, por exemplo, a linguagem javascript quando interpretada por um navegador exibirá o que contém dentro do seu código, e mostrará para o usuário o que aquela linguagem está querendo mostrar e exibirá para o usuário na tela do seu navegador assim que o navegador interpretar o código Javascript

HTML

O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações de uma página na web. Sem isso, o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos. Os hipertextos são conjuntos de elementos conectados. Esses podem ser palavras, imagens, vídeos, documento, etc. Quando conectados, formam uma rede de informações que permite a comunicação de dados, organizando conhecimentos e guardando informações.
